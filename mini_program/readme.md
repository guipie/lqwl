# 奈雪の茶小程序模板 (Vue 3 版)

> 本项目基于 [tinypuppet/nxdc-milktea](https://github.com/tinypuppet/nxdc-milktea) 升级改造，原项目基于vue2开发，且长久未更新；本项目升级为Vue 3 (Composition API)版本，将Options API 升级为 `<script setup>`风格。感谢原作者的开源贡献！


**特别声明：本项目中的展示图片归奈雪の茶所有。**

## 简介

一套仿奈雪の茶小程序的项目模板。

## 功能列表

- 首页
- 点餐（自取和外卖两种方式，有基本的点餐逻辑处理）
- 取餐
- 我的
- 积分商城
- 积分商城详情页
- 积分签到
- 会员码
- 我的卡券
- 收货地址
- 我的资料
- 我的订单
- 订单详情
- 余额
- 会员券包
- 券包详情
- 订单评价
- 买茶送包
- 更多服务
- 开发票
- 礼品卡

## 数据说明

所有数据都是静态的，保存在 `api` 文件夹下。
